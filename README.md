## Frequently Asked Questions (FAQ)

<details>
<summary>What is open-source?</summary>

Open-source refers to software whose source code is freely available for anyone to view, use, modify, and distribute. In the context of VTubersTV:
- The entire codebase is publicly accessible on GitHub
- Anyone can inspect how the platform works
- Developers can contribute improvements or fixes
- The community can verify the security and privacy measures
- No hidden proprietary code or secret algorithms
</details>

<details>
<summary>Why make this?</summary>

VTubersTV was created to address several needs in the VTuber community:
- Provide a platform specifically optimized for VTuber content
- Offer more creator-friendly monetization options
- Give creators more control over their content and data
- Create a community-driven platform that puts VTubers first
- Reduce platform dependency on large corporations
- Foster innovation in the VTuber space through open-source development
</details>

<details>
<summary>How do I contribute to the project?</summary>

There are many ways to contribute to VTubersTV:
1. **Code Contributions**
   - Fork the repository

   - Submit pull requests
   - Review other's code

2. **Non-Code Contributions**
   - Report bugs
   - Suggest features
   - Improve documentation
   - Help with translations

3. **Getting Started**
   - Read our Contributing Guidelines
