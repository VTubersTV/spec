## Overview

This document provides comprehensive guidance for optimizing network setup for streaming on VTubersTV, ensuring stable and high-quality broadcasts.

### Connection Specs

1. **Minimum Requirements**

   - Latency: <100ms

2. **Recommended Specs**
   - Upload: 20+ Mbps
   - Download: 50+ Mbps
   - Latency: <50ms
